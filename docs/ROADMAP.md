# Roadmap

## Estado atual

### Implementado inicial

- monorepo, CI e deploy do frontend;
- API Fastify, Prisma e SQLite;
- autenticação, sessões e RBAC;
- aprovação e gestão inicial de usuários;
- contas, categorias e formas de pagamento;
- receitas e despesas;
- contas a receber;
- Dashboard e Analytics iniciais.

- PostgreSQL;
- API hospedada;

## Princípio

Marcos só são concluídos quando o código existe, `npm run check` passa e a documentação corresponde ao estado real.
