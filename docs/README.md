## Regra de manutenção

Toda entrega funcional deve atualizar, quando aplicável:

- regras de negócio;
- arquitetura;
- backlog;
- roadmap;
- decisões de produto;
- documento da versão.

A documentação deve refletir o que realmente existe no repositório. Itens planejados devem ser marcados como planejados, nunca como concluídos.
