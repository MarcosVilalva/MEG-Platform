# ADR-002 — API Fastify

## Status

Aprovado.

## Contexto

O MEG precisa sair do LocalStorage e passar a usar uma camada de API.

## Decisão

Usaremos Fastify pela leveza, performance, simplicidade e boa integração com TypeScript.

## Consequências

- API rápida e simples.
- Boa base para crescer.
- Facilidade de documentação com Swagger.
