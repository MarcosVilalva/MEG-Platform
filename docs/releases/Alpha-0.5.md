# Alpha 0.5 — Cartőes e Orçamento

## Entregas
- Cadastro de cartőes com limite, fechamento e vencimento.
- Compras ŕ vista e parceladas.
- Controle de limite comprometido e disponível.
- Projeçăo de fatura por męs.
- Orçamento mensal persistente por grupo.
- Comparativo planejado, realizado e disponível.
- Isolamento por usuário e RBAC.

## Validaçăo
Executar `npm run db:generate`, `npm run db:push` e `npm run check`.
