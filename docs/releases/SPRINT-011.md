# Sprint 011 — Cadastros financeiros reais

## Entregas

- CRUD autenticado de contas bancárias.
- CRUD autenticado de categorias de receita e despesa.
- CRUD autenticado de formas de pagamento.
- Desativação lógica para preservar histórico.
- Aplicação da matriz RBAC da Sprint 010.
- Nova tela de Cadastros integrada ao menu lateral.
- Persistência por API e Prisma, sem depender de dados de exemplo.

## Permissões

- VIEWER: consulta.
- OPERATOR: consulta, criação e edição.
- MANAGER e ADMIN: consulta, criação, edição e desativação.

## Validação

Executar `npm run check`, iniciar com `npm run dev` e testar a tela **Cadastros** com a API em execução.
