# Sprint 014 — Contas a Receber

## Entregas

- Cadastro de pagadores.
- Contas a receber por usuário.
- Controle de vencimento, saldo em aberto e status.
- Recebimentos parciais ou integrais.
- Juros e multa no recebimento.
- Vínculo opcional com conta e forma de pagamento.
- Geração automática de movimentação financeira quando o recebimento é associado a uma conta.
- Indicadores de aberto, vencido e recebido.
- Restrições conforme perfil de acesso.

## Validação local

```bash
npm run db:generate
npm run db:push
npm run check
npm run dev
```

Acesse a opção **Contas a Receber** no menu lateral.
