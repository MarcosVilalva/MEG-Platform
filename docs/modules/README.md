# Módulos do Produto

| Módulo | Estado | Responsabilidade |
|---|---|---|
| Autenticação | Implementado inicial | Login, sessões e tokens |
| Usuários e permissões | Implementado inicial | Aprovação, perfis e bloqueio |
| Cadastros | Implementado inicial | Contas, categorias e formas |
| Movimentações | Implementado inicial | Receitas e despesas |
| Contas a receber | Implementado inicial | Pagadores, títulos e recebimentos |
| Dashboard | Implementado inicial | Visão rápida |
| Analytics | Implementado inicial | Comparativos e Pareto |
| Cartões | Planejado | Limites, compras, parcelas e faturas |
| Orçamento e metas | Planejado | Limites, objetivos e reserva |
| Patrimônio | Planejado | Ativos, passivos e patrimônio líquido |
| Fluxo de caixa | Parcial | Agenda e projeções |
| Notificações | Planejado | Internas, e-mail e WhatsApp |
| Importação | Planejado | OFX, CSV e XLSX |
| IA financeira | Planejado | Insights e simulações |

Cada módulo completo deve possuir persistência, API, interface, autorização, auditoria pertinente, testes e documentação.
