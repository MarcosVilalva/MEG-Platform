# MEG Financas para Android

O arquivo `MEG-Financas.apk` desta pasta e gerado automaticamente a partir da versao mais recente da aplicacao web.

O arquivo `app-version.json` informa ao aplicativo Android a versão disponível, o resumo da atualização e o SHA-256 usado para validar o APK antes da instalação.

O APK oficial é assinado por uma chave permanente guardada nos segredos do GitHub. A partir da versão 1.1, o próprio aplicativo verifica novas versões e solicita ao usuário apenas a confirmação de instalação exigida pelo Android.

Download oficial: `https://marcosvilalva.github.io/MEG-Platform/downloads/MEG-Financas.apk`
